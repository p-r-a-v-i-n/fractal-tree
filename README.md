### Fractal Tree
Implementing fractal tree in rust using raylib-rs for peace of mind.
I wanted to try something different and don't want solve any real world problem. I wrote this implementation just for break from boring web framework work.
Thats I called this tree implementation is mental peace.

you can read about fractal-tree, [here](https://en.wikipedia.org/wiki/Fractal_canopy)
